常用语句对照

  • 1.查询数据
    SELECT * FROM {pre}video 查询所有数据
    SELECT * FROM {pre}video WHERE id=1 查询指定ID数据
  • 2.删除数据
    DELETE FROM {pre}video 删除所有数据
    DELETE FROM {pre}video WHERE id=1 删除指定的第几条数据
    DELETE FROM {pre}video WHERE msg LIKE '%刘德华%' msg"刘德华"的数据
  • 3.修改数据
    UPDATE {pre}video SET status=1 将所有status字段里的值修改成"1"
    UPDATE {pre}video SET status=1 WHERE id=1 指定的第几条数据把status字段里的值修改成"1"
  • 4.替换视频地址
    UPDATE {pre}video SET src=REPLACE(src, '原始字符串', '替换成其他字符串')
  • 5.清空数据ID重新从1开始
    TRUNCATE {pre}video
